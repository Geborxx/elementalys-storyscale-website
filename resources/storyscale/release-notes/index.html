Current Status

StoryScale is in development/release-candidate validation. The Forge app has been deployed to a Jira development environment and is being prepared for field testing and Marketplace readiness work.

Notable Capabilities

  • Jira project page app built on Atlassian Forge.
  • Sprint and JQL imports, plus single issue lookup.
  • Readiness review with local rule-based signals.
  • Comparative estimation board with hidden Story Points during comparison.
  • Historical anchor calibration and guided calibration fallback.
  • Session snapshots in Forge Storage with resume and delete actions.
  • Controlled publish to Jira Story Points after confirmation.

Before Marketplace Release

  • Field test on real Jira sprint data.
  • Validate publish behavior on a small controlled sprint.
  • Complete release visual QA and admin-facing documentation.
  • Expand automated tests around calibration, publish, and resolver behavior.
